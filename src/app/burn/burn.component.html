Burn address

A burn address is not real burning but rather locking tokens forever.

Burn address: addr1wxa7ec20249sqg87yu2aqkqp735qa02q6yd93u28gzul93ghspjnt

Verify the address

To verify that this address is really locked forever save this to a file named locked_script.json 

{
  "type": "all",
  "scripts":
  [
    {
      "type": "before",
      "slot": 0
    }
  ]
}

Then generate the address from it 

cardano-cli address build --payment-script-file locked_script.json --mainnet --out-file locked.addr
cat locked.addr

It results in addr1wxa7ec20249sqg87yu2aqkqp735qa02q6yd93u28gzul93ghspjnt

Since the script validates only when the current slot is before 0 it will never be able to succeed.
